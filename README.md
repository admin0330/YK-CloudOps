# ym1r

`ym1r` 是一个以运维为核心的个人网站：主页展示运维中心、服务器状态和 AI 运维助手，个人主页与项目内容则收在次级入口中。

线上地址：
- [https://ym3861.cn](https://ym3861.cn)
- [https://www.ym3861.cn](https://www.ym3861.cn)

ICP备案：
- [湘ICP备2026017602号](https://beian.miit.gov.cn/)

## 站点定位

- 首页优先展示服务器运维、权限审计、日志分析和 AI 辅助排障
- 个人主页、项目页和 GitHub 被整理到单独的次级入口
- AI 只作为运维助手，不作为主入口
- 整站采用更克制的 Apple / Liquid Glass 风格

## 页面结构

### `/`

- 运维介绍页
- 提供运维中心的首屏展示
- 强调服务器管理、权限审计和 AI 运维助手

### `/portal`

- 个人主页与项目入口页
- 承接个人介绍、项目展示和 GitHub 链接

### `/cloudops`

- AI 运维助手
- 适合日志解释、故障排查和命令建议

### `/admin`

- 管理界面
- 用于用户、文件、服务器和后台数据管理

## 设计语言

- Apple 风格的留白和层级
- Liquid Glass 风格按钮与卡片
- 双语切换统一生效
- 桌面与移动端都做了布局适配
